| **–ó–∞–≥–æ–ª–æ–≤–∫–∏ –Ω–∞ –µ—Å—Ç–µ—Å—Ç–≤–µ–Ω–Ω–æ–º —è–∑—ã–∫–µ** (`Role:`, `Instructions:`) | –ò—Å–ø–æ–ª—å–∑—É–µ—Ç –æ–±—ã—á–Ω—ã–π —è–∑—ã–∫ –¥–ª—è –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏—è —Å–µ–∫—Ü–∏–∏. | - –ù–∞–∏–±–æ–ª–µ–µ —É–¥–æ–±–Ω–æ –¥–ª—è —á–µ–ª–æ–≤–µ–∫–∞.<br>- –ù–µ—Ç –∫–æ–Ω—Ñ–ª–∏–∫—Ç–æ–≤ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–∞. | - –ú–æ–¥–µ–ª—å –º–æ–∂–µ—Ç –ª–µ–≥–∫–æ –ø—Ä–æ–∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å –∏–ª–∏ –Ω–µ–≤–µ—Ä–Ω–æ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞—Ç—å.<br>- –¢—Ä—É–¥–Ω–æ –ø–∞—Ä—Å–∏—Ç—å –ø—Ä–æ–≥—Ä–∞–º–º–Ω–æ. | –ü—Ä–æ—Å—Ç—ã–µ –æ–¥–Ω–æ–∫—Ä–∞—Ç–Ω—ã–µ –ø—Ä–æ–º–ø—Ç—ã, –≥–¥–µ –≥–ª–∞–≤–Ω–æ–µ ‚Äî —É–¥–æ–±—Å—Ç–≤–æ —á—Ç–µ–Ω–∏—è —á–µ–ª–æ–≤–µ–∫–æ–º. |
| **–°—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ** (JSON, YAML) | –í–µ—Å—å –ø—Ä–æ–º–ø—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω –≤ –≤–∏–¥–µ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞. | - –ú–∞—à–∏–Ω–æ—á–∏—Ç–∞–µ–º–æ—Å—Ç—å –ø–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—é.<br>- –ò–¥–µ–∞–ª—å–Ω–æ –¥–ª—è –æ–±–º–µ–Ω–∞ –º–µ–∂–¥—É –∞–≥–µ–Ω—Ç–∞–º–∏.<br>- –ì–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç—Å—è —Å—Ö–µ–º–∞. | - –ü–ª–æ—Ö–æ —á–∏—Ç–∞–µ—Ç—Å—è –∏ –ø–∏—à–µ—Ç—Å—è —á–µ–ª–æ–≤–µ–∫–æ–º.<br>- –ù–µ –≤—Å–µ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã LLM —Å —ç—Ç–∏–º —Ä–∞–±–æ—Ç–∞—é—Ç. | **–ò–¥–µ–∞–ª—å–Ω—ã–π –≤–∞—Ä–∏–∞–Ω—Ç –¥–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö –º–µ–∂–¥—É –∞–≥–µ–Ω—Ç–∞–º–∏ –≤–∞—à–µ–π —Å–∏—Å—Ç–µ–º—ã.** |

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –±–æ–ª—å—à–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–∞ —Ç–æ, –∫–∞–∫–æ–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –¥–ª—è —Ä–∞–∑–º–µ—Ç–∫–∏ –ø—Ä–æ–º–ø—Ç–æ–≤ –≤—ã–±—Ä–∞—Ç—å, –æ–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ñ–∞–∫—Ç–æ—Ä–æ–≤:
1. –¶–µ–ª—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è - –¥–ª—è —á–∞—Ç–∞ –∏–ª–∏ –º–µ–∂–∞–≥–µ–Ω—Ç–Ω–æ–≥–æ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è.
1. –û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏ –æ–±—É—á–µ–Ω–∏—è –¥–∞–Ω–Ω–æ–π –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–π –º–æ–¥–µ–ª–∏ - —Å–º–æ—Ç—Ä–µ—Ç—å —Ä–µ–∫–æ–º–µ–Ω–¥–∞—Ü–∏–∏ –≤–µ–Ω–¥–æ—Ä–æ–≤.

### 2.2 Internal Agent Prompting vs Inter-Agent Communication

–ù–µ–æ–±—Ö–æ–¥–∏–º–æ —Ä–∞–∑–¥–µ–ª—è—Ç—å –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏ *–≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–≥–æ –ø—Ä–æ–º–ø—Ç–∏–Ω–≥–∞ –∞–≥–µ–Ω—Ç–∞* (*internal agent prompting*) –∏ *–º–µ–∂–∞–≥–µ–Ω—Ç—Å–∫–æ–π –∫–æ–º–º—É–Ω–∏–∫–∞—Ü–∏–∏* (*inter-agent communication*).

#### Internal Agent Prompting: –°—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∞—Ü–∏—è —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª–µ–π –¥–ª—è —è—Å–Ω–æ—Å—Ç–∏ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–≥–æ –ø—Ä–æ–º–ø—Ç–∞

–î–ª—è –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏—Ö –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π *–≤ —Ä–∞–º–∫–∞—Ö* –ø—Ä–æ–º–ø—Ç–∞ –æ–¥–Ω–æ–≥–æ –∞–≥–µ–Ω—Ç–∞ (system prompt) –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –ø–∞—Ç—Ç–µ—Ä–Ω `### SECTION ###`.

OpenAI [—Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç](https://help.openai.com/en/articles/6654000-best-practices-for-prompt-engineering-with-the-openai-api) –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª–∏, —Ç–∞–∫–∏–µ –∫–∞–∫ `###`, –¥–ª—è –æ—Ç–¥–µ–ª–µ–Ω–∏—è –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π –æ—Ç –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞. –ö–æ–Ω–∫—Ä–µ—Ç–Ω—ã–π —à–∞–±–ª–æ–Ω `### SECTION ###` ‚Äî —ç—Ç–æ —Ä–∞–∑—Ä–∞–±–æ—Ç–∞–Ω–Ω–æ–µ —Å–æ–æ–±—â–µ—Å—Ç–≤–æ–º —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ —ç—Ç–æ–≥–æ –ø—Ä–∏–Ω—Ü–∏–ø–∞, –∫–æ—Ç–æ—Ä–æ–µ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –µ—â—ë –±–æ–ª–µ–µ —á—ë—Ç–∫–∏–µ —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏–µ –≥—Ä–∞–Ω–∏—Ü—ã –¥–ª—è —Å–ª–æ–∂–Ω—ã—Ö –∑–∞–ø—Ä–æ—Å–æ–≤. 

> –ï–≥–æ –æ—Å–Ω–æ–≤–Ω–æ–µ –ø—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–æ ‚Äî –≤—ã—Å–æ–∫–∞—è –≤–∏–¥–∏–º–æ—Å—Ç—å –∏ –º–∞–ª–æ–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–ø—É—Ç–∞—Ç—å —Å —Å–æ–¥–µ—Ä–∂–∏–º—ã–º. –û–Ω —Å–æ–∑–¥–∞–µ—Ç —á—ë—Ç–∫–∏–µ –≤–∏–∑—É–∞–ª—å–Ω—ã–µ –∏ —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏–µ –≥—Ä–∞–Ω–∏—Ü—ã –¥–ª—è LLM, –ø–æ–º–æ–≥–∞—è –æ—Ç–¥–µ–ª—è—Ç—å —Ä–æ–ª—å, –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –≤–≤–æ–¥, –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ –∏ —Ñ–æ—Ä–º–∞—Ç –≤—ã–≤–æ–¥–∞. –ö–æ–Ω—Ñ–ª–∏–∫—Ç —Å Markdown –Ω–µ –≤–∞–∂–µ–Ω, —Ç–∞–∫ –∫–∞–∫ –ø—Ä–æ–º–ø—Ç –Ω–µ –æ—Ç–æ–±—Ä–∞–∂–∞–µ—Ç—Å—è –≤ Markdown, –∞ —è–≤–ª—è–µ—Ç—Å—è —á–∏—Å—Ç—ã–º —Ç–µ–∫—Å—Ç–æ–º.
    
#### –ú–µ–∂–∞–≥–µ–Ω—Ç–Ω–æ–µ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–µ: –ü—Ä–∏–º–µ–Ω–µ–Ω–∏–µ —Å—Ç—Ä–æ–≥–æ–≥–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ –¥–∞–Ω–Ω—ã—Ö –¥–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ –º–µ–∂–¥—É –∞–≥–µ–Ω—Ç–∞–º–∏

–î–ª—è –ª—é–±–æ–≥–æ –≤—ã–≤–æ–¥–∞ –æ–¥–Ω–æ–≥–æ –∞–≥–µ–Ω—Ç–∞, –∫–æ—Ç–æ—Ä—ã–π —Å—Ç–∞–Ω–µ—Ç –≤–≤–æ–¥–æ–º –¥–ª—è –¥—Ä—É–≥–æ–≥–æ, –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ —Å—Ç—Ä–æ–≥—É—é —Å—Ö–µ–º—É –≤—ã–≤–æ–¥–∞ –≤ —Ñ–æ—Ä–º–∞—Ç–µ YAML/JSON. –≠—Ç–æ —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è "–∫–æ–Ω—Ç—Ä–∞–∫—Ç–æ–º" –º–µ–∂–¥—É –∞–≥–µ–Ω—Ç–∞–º–∏.

> –í –ø—Ä–æ–º–ø—Ç–µ –∞–≥–µ–Ω—Ç—É —É–∫–∞–∑—ã–≤–∞–µ—Ç—Å—è: "*–í–∞—à –∏—Ç–æ–≥–æ–≤—ã–π –≤—ã–≤–æ–¥ –î–û–õ–ñ–ï–ù –±—ã—Ç—å –≤–∞–ª–∏–¥–Ω—ã–º JSON-–æ–±—ä–µ–∫—Ç–æ–º —Å–æ–≥–ª–∞—Å–Ω–æ —Å–ª–µ–¥—É—é—â–µ–π —Å—Ö–µ–º–µ: {...}*". –ü—Ä–∏ —ç—Ç–æ–º —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª–∏ `###` –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è *–≤–Ω—É—Ç—Ä–∏ —Å–∞–º–æ–≥–æ –ø—Ä–æ–º–ø—Ç–∞* –¥–ª—è –æ—Ç–¥–µ–ª–µ–Ω–∏—è —Å–µ–∫—Ü–∏–π –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π –¥–ª—è –∞–≥–µ–Ω—Ç–∞.

**–ü—Ä–æ–±–ª–µ–º—ã:**
	- –ó–∞—Ç—Ä–∞—Ç—ã –Ω–∞ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é: –≤—ã—Å–æ–∫–∏–µ. –¢—Ä–µ–±—É–µ—Ç—Å—è —Ç—â–∞—Ç–µ–ª—å–Ω–∞—è —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∞ JSON-—Å—Ö–µ–º –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —Ç–∏–ø–∞ –≤—ã–≤–æ–¥–∞ –∞–≥–µ–Ω—Ç–∞.
	- –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ —Ç–æ–∫–µ–Ω–æ–≤: –≤–æ–∑–º–æ–∂–Ω–æ, –≤—ã—à–µ, –Ω–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –∑–Ω–∞—á–∏—Ç–µ–ª—å–Ω–æ —Å–Ω–∏–∂–∞–µ—Ç –æ—à–∏–±–∫–∏ –ø–∞—Ä—Å–∏–Ω–≥–∞ –∏ –≥–∞–ª–ª—é—Ü–∏–Ω–∞—Ü–∏–∏.
	- –ù–æ–≤—ã–µ —Ä–∏—Å–∫–∏: –∞–≥–µ–Ω—Ç –º–æ–∂–µ—Ç –≤—ã–¥–∞—Ç—å –Ω–µ–≤–∞–ª–∏–¥–Ω—ã–π JSON, –ø–æ—ç—Ç–æ–º—É –Ω—É–∂–µ–Ω —Å–ª–æ–π –≤–∞–ª–∏–¥–∞—Ü–∏–∏ –≤ –∫–æ–¥–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –º–æ–¥–µ–ª—å Pydantic –≤ Python) –¥–ª—è –ø—Ä–æ–≤–µ—Ä–∫–∏ –∏ –ø–æ–≤—Ç–æ—Ä–Ω–æ–≥–æ –∑–∞–ø—É—Å–∫–∞.
	
–≠—Ç–æ —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω –º–Ω–æ–≥–æ–∞–≥–µ–Ω—Ç–Ω—ã—Ö –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä. –û–Ω –æ–±—Å—É–∂–¥–∞–µ—Ç—Å—è –≤ —Ä–µ—Å—É—Ä—Å–∞—Ö –ø–æ **Agent Frameworks** –≤—Ä–æ–¥–µ AutoGen –∏ [CrewAI](https://docs.crewai.com/core-concepts/Tasks/).

**–¢–∞–∫–æ–π –ø–æ–¥—Ö–æ–¥ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç:**

1. **–ò–Ω—Ç–µ–≥—Ä–∞—Ü–∏—é —Å —Å–∏—Å—Ç–µ–º–æ–π –∫–æ–Ω—Ç—Ä–æ–ª—è –≤–µ—Ä—Å–∏–π (Git)**
   - –ö–æ–≥–¥–∞ –ø—Ä–æ–º–ø—Ç—ã —Ö—Ä–∞–Ω—è—Ç—Å—è –≤ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö —Ñ–∞–π–ª–∞—Ö, Git –ª–µ–≥–∫–æ –º–æ–∂–µ—Ç —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å –∏—Ö –∏ –æ—Ç—Å–ª–µ–∂–∏–≤–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤–æ –≤—Ä–µ–º–µ–Ω–∏.
   - –ü—Ä–∏–º–µ—Ä: –µ—Å–ª–∏ –∑–∞–≤—Ç—Ä–∞ –≤—ã –∏–∑–º–µ–Ω–∏—Ç–µ ‚Äútemperature‚Äù —Å 0.3 –Ω–∞ 0.4, diff –≤ Git –ø–æ–∫–∞–∂–µ—Ç –∏–º–µ–Ω–Ω–æ —ç—Ç–æ ‚Äî –ø–æ–ª–µ–∑–Ω–æ –¥–ª—è –∞—É–¥–∏—Ç–∞ –∏ –≤–æ—Å–ø—Ä–æ–∏–∑–≤–æ–¥–∏–º–æ—Å—Ç–∏.
2. **–ú–∞—à–∏–Ω–Ω–∞—è —á–∏—Ç–∞–µ–º–æ—Å—Ç—å –∏ –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –≤–∞–ª–∏–¥–∞—Ü–∏—è**
