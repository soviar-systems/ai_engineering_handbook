**Изменения в файловой структуре сайта** https://book.soviar.ru/

В репозитории https://github.com/lefthand67/ai_engineering_handbook, где ведутся исходники сайта, произошел рефакторинг:

- содержимое директории `helpers` переехало в `tools`, так как по смыслу эти директории очень похожи. 

В `helpers` лежали вспомогательные промты и скрипты по автоматизации работы с файлами, а также инструкции и конфиги по настройке веб-сайта. В `tools` лежала документация по использованию инструментов разработчика в данном репозитории, например, `jupytext`.

Теперь все инструкции лежат в отдельной директории `tools/docs`, а скрипты и тесты к ним в своих директориях:

```bash
$ tree -dL 2 --gitignore tools
tools
├── docs
│   ├── ai_agents
│   ├── git
│   ├── jupyter_and_markdown
│   ├── languages
│   ├── scripts_instructions
│   └── website
├── prompt_helpers
├── scripts
└── tests

11 directories
```

Поэтому если некоторые ссылки в постах выше не будут работать, значит, статьи сменили имя в связи с переездом в новое месторасположение. Репозиторий развивается, поэтому такой рефакторинг, к сожалению, неизбежен.
